# Support CRM

Support CRM is a customer support platform designed to help support teams manage tickets, conversations, customers, and service performance from one place.

## Goals
- Centralize customer support requests and communication
- Route issues to the right team or agent quickly
- Track SLA compliance and response times
- Provide a simple reporting layer for service quality
